Research Interests

  • Models of neuromorphic computation
  • Parallel functional programming, cost models
    • AJITPar project (2013 - 2017)
  • Parallel symbolic computation
    • HPC-GAP project (2009 - 2013)
  • Resource analysis and control
    • MOBIUS project (2006 - 2009)
  • Temporal logic, model checking, program analysis
  • Compositional verification, assume-guarantee reasoning
